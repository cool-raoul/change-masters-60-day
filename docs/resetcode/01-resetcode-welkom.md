Programma's
De Reset Code is geen dieet en geen snelle oplossing. Het is een holistische reis waarin je stap voor stap ontdekt hoe jouw lichaam weer kan gaan werken zoals het behoort te werken. Dat vraagt tijd, aandacht en liefde voor jezelf.

DE RESET CODE bestaat uit 2 stappen:

1️⃣ Het 16 daagse Darmprogramma:
Dit is de eerste stap. Veel mensen ervaren tijdens deze periode dat ze bewuster met voeding omgaan en merken hoe fijn het is om zich lichter, energieker en helderder te voelen.

2️⃣ De Holistic Reset:
Drie keer 21 dagen waarin je verder verdiept, meer inzicht krijgt en ontdekt hoe krachtig het kan zijn om je leefstijl te veranderen en te voelen wat het voor je doet.
